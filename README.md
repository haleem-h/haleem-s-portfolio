## 📌 Overview

This is my personal portfolio website, built using an AI-assisted development approach.

Coming from a non-web development background, I created this project using a workflow I call **“vibe coding”** — where ideas are translated into reality through structured thinking, precise prompt engineering, and iterative refinement using AI tools.

Instead of traditional coding from scratch, I focused on:

* Defining clear requirements
* Writing accurate and detailed prompts
* Iterating on outputs to improve design and functionality

This project demonstrates how powerful AI tools can be when used effectively to build real-world applications.
